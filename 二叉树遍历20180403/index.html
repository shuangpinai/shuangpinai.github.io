二叉树遍历

发表于 2018-04-03 | 分类于 java

主要介绍二叉树的先、中、后序遍历，使用递归和栈两种实现方式

二叉树的3中遍历方式

  • 先序遍历：先跟节点遍历，再依次遍历左子树和右子树
  • 中序遍历：先遍历左子树，再遍历跟节点，最后遍历右子树
  • 后序遍历：先依次左子树和右子树，最后遍历跟节点

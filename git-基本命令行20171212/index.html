git reset 共有3种模式，分别是

  • git reset –mixed 保留现有工作区，但是会用指定的commit覆盖暂存区。

  • git reset –soft 保留暂存区，但是会清空工作区。

  • git reset –hard 简单粗暴，直接使用指定的commit的内容覆盖stage区以及工作区。
